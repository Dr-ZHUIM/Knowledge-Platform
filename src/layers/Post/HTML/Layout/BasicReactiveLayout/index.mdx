### 1. Flex 布局 与 Grid 布局

在 HTML5 的时代，前端开发者们往往使用 flex 弹性盒子布局与 grid 栅格布局来构建复杂的布局 (Layout)。

相比较来说，flex 布局倾向单主轴，多副轴的布局，grid 布局倾向多主轴,多副轴<Sup title="grid布局的主轴为行级格式上下文轴，由writing-mode属性操作" />的布局，从其的一些基本属性就能看出端倪：

- flex 基本案例

```scss
// scss
.flex-example {
  display: flex;
  // 选择主轴
  flex-direction: column;
  // 操作整条主轴布局
  justify-content: center;
  // 操作所有的副轴布局
  align-items: center;
  // 副轴单独元素操作
  .cross-item {
    align-self: flex-start;
    // flex布局的主轴元素无法单独布局，justify-self 不可用
  }
}
```

- grid 基本案例

```scss
.grid-example {
  display: grid;
  grid-template-rows: repeat(3, 1fr);
  grid-template-columns: repeat(4, 1fr);
  /* 操作所有的主轴布局 */
  justify-content: center;
  /* 操作所有的副轴布局 */
  align-items: center;
  /* 副轴单独元素操作 */
  .cross-item {
    justify-self: center;
    align-self: center;
  }
}
```
